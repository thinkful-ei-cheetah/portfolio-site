Laconic

          

            Laconic is built for YouTube content creators. How successful a video can be is hugely dependent on how well it stands out within search results.  That means having an eye catching thumbnail, title and description.  Laconic allows creators to preview how their video will look in search results, on both mobile and desktop, before publishing.  Key concepts learned during development include component composition, nested routes and api caching.

Bonjour!

          

            Bonjour! is a French language learning tool that implements the spaced repetition method for improved memory recall. The app is styled minimally and is a proof of concept.  The app is built using React, Node and Postgres and is deployed via Heroku for the api and Zeit for the client.  Key concepts learned include using a singly linked list and queue to implement the space repetition protocol.

Stockpile

          

            Stockpile is a small app that allows anyone to easily save and rate bookmarks for interesting links they find on the web.  The app is fully responsive and uses no frameworks or libraries outside of Jquery to work.  Key concepts learned during the development include IFEE functions to encapsulate logic, the handler - data - view pattern,and how to successfully handle promise chaining while working with an external api.
